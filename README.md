alcohol-supply-chain-blockchain
Blockchain Alcohol Supply Chain Tracker

Overview
This system uses Ethereum blockchain to track alcohol batches from production to final consumer.

 Features
- Batch creation (only authorized producers)
- Ownership transfer tracking
- Full transparency and traceability
- Immutable blockchain records

Tools Used
- Solidity
- Remix IDE
- MetaMask
- Sepolia Testnet

How to Run
1. Open Remix IDE
2. Paste smart contract
3. Compile and deploy
4. Add yourself as producer
5. Create batch
6. Transfer ownership
7. Retrieve batch details
